# Smart-Pneumatic-Agro-Sorting-System
PLC-HMI based industrial automation system for pneumatic object sorting using Siemens S7-1200 and WinCC HMI.
Smart Pneumatic Agro Sorting System

Overview

A PLC-HMI based industrial automation system developed using Siemens TIA Portal V16 for automated object sorting using pneumatic actuation.

Features

- Automatic and Manual Operation Modes
- Sensor-Based Object Detection
- Pneumatic Cylinder Control
- PLC Ladder Logic Programming
- WinCC HMI Interface
- Alarm Monitoring System
- Batch Counting and Reset Logic
- Real-Time Operator Control

Hardware Components

- Siemens S7-1200 PLC
- Pneumatic Cylinder
- Solenoid Valve
- Object Detection Sensor
- HMI Panel (WinCC)

Software Used

- Siemens TIA Portal V16
- Ladder Logic (LAD)
- WinCC HMI

Working Principle

1. Object detected by sensor.
2. PLC processes input signal.
3. Timer logic activates pneumatic cylinder.
4. Object is sorted.
5. Counter updates production count.
6. Alarm triggers during fault conditions.
7. HMI displays system status and controls.

- Automated Sorting Operation
